Ajax (Async/XML) Request- communicate by making http request/no need to reload page

Ajax short for "Asynchronous JavaScript And XML")[1][2] is a set of Web development techniques using many web technologies
 on the client side to create asynchronous Web applications.
 With Ajax, web applications can send and retrieve data from a server asynchronously
 (in the background) without interfering with the display and behavior of the existing page.
